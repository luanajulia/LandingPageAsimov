# 🐍 Python IA Academy — Landing Page High-End

Esta é uma Landing Page de alta performance desenvolvida com **React** e **Tailwind CSS v4**, focada na conversão para um curso de Python especializado em Inteligência Artificial. O design segue uma estética *Premium Dark*, com componentes interativos e modais de conversão.

## 🤖 Desenvolvimento Orientado por IA

Este projeto foi um exercício prático de **IA-Augmented Development**, onde a colaboração entre humano e inteligência artificial acelerou o ciclo de desenvolvimento e a resolução de problemas complexos de infraestrutura.

### 🛠️ Quais ferramentas de IA foram utilizadas?
* **Gemini (Google):** Atuou como parceiro de *pair programming*, auxiliando na arquitetura de componentes, lógica de estados do React (`useState`) e correção de erros de build.
* **Tailwind Engine:** Utilizado para a geração dinâmica de classes utilitárias e estruturação do Design System.

### 🚀 Em que etapas a IA ajudou mais?
* **Resolução de Erros de Ambiente:** A IA foi crucial para diagnosticar e corrigir o conflito de versões entre o **PostCSS** e o **Tailwind v4**, configurando corretamente o arquivo `postcss.config.js` para que o Vite pudesse compilar o CSS.
* **Estruturação de Componentes:** Ajuda na criação de componentes reutilizáveis e escaláveis, como o sistema de **Modais Dinâmicos** (Cadastro e Ementa) e o **Hero Section** animado.
* **Design e Estilização:** Sugestão de paletas de cores (Dark Mode com acentos em roxo `#A855F7`) e efeitos visuais como o *Glassmorphism* e o terminal interativo.

### ✍️ O que foi ajustado manualmente e porquê?
* **Tipografia e Textos:** Alterei manualmente as fontes e refinei o *copy* (frases e chamadas) para garantir uma comunicação clara e persuasiva, alinhada ao público de tecnologia.
* **Espaçamento e Pacing (Paddings):** Ajustei minuciosamente os paddings e margens dos textos e seções. O objetivo foi garantir um respiro visual adequado, evitando que o conteúdo parecesse apertado e melhorando a legibilidade.
* **Estilização e Style de Componentes:** Personalizei o estilo dos componentes React, aplicando sombras suaves e bordas arredondadas específicas para manter a consistência visual em toda a página.
* **Gestão de Assets e Favicon:** Configurei manualmente o ícone do site (o formato "X" arredondado) na pasta `public` do Vite para evitar problemas de cache e garantir a exibição correta em todos os navegadores.
* **Refinamento de UX:** Ajustei as propriedades de desfoque de fundo (`backdrop-blur`) e a opacidade dos modais para criar uma hierarquia visual clara quando uma ação é disparada pelo usuário.
  
## 🛠️ Tecnologias Utilizadas

- **Vite** (Build Tool)
- **Tailwind CSS v4**
