## 🤖 Desenvolvimento Orientado por IA
### 🛠️ Quais ferramentas de IA foram utilizadas?
* **Gemini (Google) & Tailwind Engine** 
### 🚀 Em que etapas a IA ajudou mais?
* **Resolução de Erros de Ambiente:** A IA foi crucial para diagnosticar e corrigir o conflito de versões entre o **PostCSS** e o **Tailwind v4**, configurando corretamente o arquivo `postcss.config.js` para que o Vite pudesse compilar o CSS.* **Estruturação de Componentes:** Ajuda na criação base dos componentes e fui personalizando cada elemento como o de referencia.
### ✍️ O que foi ajustado manualmente e porquê?
* **Estilização Avançada de Componentes:** Alterei manualmente os fontes e refinei o codigo base fornecido pela IA, fui além das classes padrão do Tailwind, criando uma identidade visual única através de:
* **Glow Effects:** Implementação de sombras internas e externas customizadas (glow roxo) para dar profundidade aos botões e cards.
* **Geometria de Interface:** Personalização rigorosa de raios de borda (border-radius) e paddings internos para alcançar o visual Neo-brutalista Moderno que o projeto exigia.
* **Hierarquia de Cores:** Ajuste manual de gradientes e opacidades (white/5, purple/20) para garantir que o Dark Mode fosse confortável aos olhos, mantendo o contraste necessário
